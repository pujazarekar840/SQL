1.String Concatenation in Java:
-In java, string concatenation forms a new string that is the combination of multiple strings. There are two ways to concat 
 string in java:
-String concatenation is implemented through the StringBuilder (or StringBuffer) class and its append method. String 
 concatenation operator produces a new string by appending the second operand onto the end of the first operand. The string 
 concatenation operator can concat not only string but primitive values also.
-After a string literal, all the + will be treated as string concatenation operator.

A. By + (string concatenation) operator
B. By concat() method


2. Eg:1
class TestStringConcatenation1{  
 public static void main(String args[]){  
   String s="Sachin"+" Tendulkar";  
   System.out.println(s);//Sachin Tendulkar  
 }  
}  

Eg:2
class TestStringConcatenation2{  
 public static void main(String args[]){  
   String s=50+30+"Sachin"+40+40;  
   System.out.println(s);//80Sachin4040  
 }  
}

Eg:3
class TestStringConcatenation3{  
 public static void main(String args[]){  
   String s1="Sachin";  
   String s2="Tendulkar";  
   String s3=s1.concat(s2);  
   System.out.println(s3);//Sachin Tendulkar  
  }  
} 



2. Substring in Java:
- A part of string is called substring. In other words, substring is a subset of another string. In case of substring 
  startIndex is inclusive and endIndex is exclusive(take the first and leave the last).
- index starts from zero.

Eg:
public class TestSubstring{  
 public static void main(String args[]){  
   String s="SachinTendulkar";  
   System.out.println(s.substring(6));//Tendulkar  
   System.out.println(s.substring(0,6));//Sachin  
 }  
}
