1.Java String class methods:
- The java.lang.String class provides a lot of methods to work on string. By the help of these methods, we can perform 
  operations on string such as trimming, concatenating, converting, comparing, replacing strings etc.Java String is a 
  powerful concept because everything is treated as a string if you submit any form in window based, web based or mobile 
  application.

2.1 Java String toUpperCase() and toLowerCase() method:
Eg:1
public class Testmethodofstringclass{
 public static void main(String args[]){
 
   String s="Sachin";
   System.out.println(s.toUpperCase());//SACHIN
   System.out.println(s.toLowerCase());//sachin
   System.out.println(s);//Sachin(no change in original)
 }
}

2.6 Java String intern() method:
Eg:
public class Testmethodofstringclass5{
 public static void main(String args[]){
 
   String s=new String("Sachin");
   String s2=s.intern();
   System.out.println(s2);//Sachin
  }
}

Note:A pool of strings, initially empty, is maintained privately by the class String.
When the intern method is invoked, if the pool already contains a string equal to this String object as determined by the 
equals(Object) method, then the string from the pool is returned. Otherwise, this String object is added to the pool and a
 reference to this String object is returned.

2.8 Java String replace() method:
Eg:
   public class Testmethodofstringclass5{
    public static void main(String args[]){
      String s1="Java is a programming language. Java is a platform. Java is an Island.";    
      String replaceString=s1.replace("Java","Kava");//replaces all occurrences of "Java" to "Kava"    
      System.out.println(replaceString);    
    }
  }


note:The string replace() method replaces all occurrence of first sequence of character with second sequence of character.
